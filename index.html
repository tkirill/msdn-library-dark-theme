    • Dark theme for MSDN Library

    Third-party user style implementing dark theme for MSDN Library.

    Project maintained by tkirill Hosted on GitHub Pages — Theme by mattgraham

    For those who read documentation a lot. It is implemented using user style and the colors are same as in the dark theme from Microsoft Visual Studio 2013.

    Installation

    Firefox

    1. Install Stylish extension bringing user styles into browser.
    2. Visit our page on userstyles.org: userstyles.org/styles/109048/msdn-library-dark-theme.
    3. Press "Install with Stylish" button.
    4. For update go to Stylish -> Manage styles then right click on style and click "Find updates".

    Chrome

    Same as Firefox. Stylish for Chrome: chrome.google.com/webstore/detail/stylish/fjnbnpbmkenffdnngjfgmeleoegfcffe
